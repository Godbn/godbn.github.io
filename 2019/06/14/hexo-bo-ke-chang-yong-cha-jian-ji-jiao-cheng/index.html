Hexo博客常用插件及教程

本文最后更新于：2020年7月29日 下午

为大家总结下hexo常用的插件与教程
合理的使用插件能够提高hexo的效率

命令 功能 文档
npm install hexo-generator-index-pin-top –save
npm uninstall hexo-generator-index –save		 文章置顶插件 传送门
npm i –save hexo-wordcount 文章字数+阅读时长统计 传送门
npm install hexo-baidu-url-submit –save 百度链接主动推送 传送门
npm install hexo-deployer-ftpsync –save hexo上传至FTP服务器 传送门
npm install hexo-deployer-cos –save hexo上传至腾讯云COS 传送门
npm install hexo-deployer-aliyun –save hexo上传至阿里云 传送门
npm install –save hexo-admin
(如何做双分支备份最好不要用)		 hexo管理插件 传送门
hexo-blog-encrypt(请看文档) hexo文章加密码 传送门
卜算子ip计数统计 IP计数统计 传送门
npm install leancloud-storage –save Valine评论 传送门
Valine评论邮件通知 Valine评论提醒 传送门
npm install hexo-helper-live2d –save Live2D看板喵 传送门
