主题maupassant增加canvas_nest动态粒子效果和不蒜子统计

这个动态粒子效果想搞很久了，但是这个主题给出的配置不行

我是在google上搜索才找到的，某度上搜索的信息太少了

因为某度对某些网站不爬取 所以搜索不到

附上方法

用Notepad++打开文件footer.pug，其他编辑器都可以，能修改就行

文件位置: themes\maupassant\layout\_partial\footer.pug

在下面添加一个script标签就可以，主题下_config.yml里面的不需要设置

script(type="text/javascript",color="240,128,128",opacity='0.9',zIndex="-2",count="60",src="//cdn.bootcss.com/canvas-nest.js/1.0.1/canvas-nest.min.js")

同理不蒜子也是在这个文件里面添加

script(async='', src='//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js')
span#busuanzi_container_site_pv
  | 本站总访问量
  span#busuanzi_value_site_pv
  | 次

附上我配置的footer.pug

#footer= 'Copyright © ' + '2018 - ' + date(Date.now(), 'YYYY') + ' ' + '| '
  a(href=url_for('.'), rel='nofollow')= config.title  + ' | '
  | 豫ICP备16011765号-1 <br>
  <!-- 不蒜子统计 -->
  script(async='', src='//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js')
  span#busuanzi_container_site_pv
  | 本站总访问量
  span#busuanzi_value_site_pv
  | 次
  <!-- 动态粒子效果 -->
  script(type="text/javascript",color="255,23,0",opacity='0.8',zIndex="-2",count="80",src="//cdn.bootcss.com/canvas-nest.js/1.0.1/canvas-nest.min.js")

结尾

这样就可以很ok了

2019.06.26 追加

增加百度自动推送

在主题目录下创建文件 source\js\baidupush.js 输入以下代码

(function(){
var bp = document.createElement('script');
var curProtocol = window.location.protocol.split(':')[0];
if (curProtocol === 'https') {
    bp.src = 'https://zz.bdstatic.com/linksubmit/push.js';        
}
else {
    bp.src = 'http://push.zhanzhang.baidu.com/push.js';
}
var s = document.getElementsByTagName("script")[0];
s.parentNode.insertBefore(bp, s);
})();

在 head.pug 上面添加出错，只能添加到 footer.pug 文件上


<!-- 百度自动推送 -->
script(type='text/javascript', src=url_for(theme.js) + '/baidupush.js')
